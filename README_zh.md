# v2rayB

[**English**](README.md)&nbsp;&nbsp;&nbsp;[**简体中文**](README_zh.md)

v2rayB 是 [v2rayA](https://github.com/v2rayA/v2rayA) 的分支版本 - 一个支持全局透明代理的 V2Ray 客户端，同时兼容 SS、SSR、Trojan(trojan-go)、Tuic 与 [Juicity](https://github.com/juicity)协议。 [[SSR支持清单]](https://github.com/v2rayA/dist/shadowsocksR/blob/master/README.md#ss-encrypting-algorithm)

## v2rayB 新特性

- 受 WebSocket logo 启发的深色 + 黄色主题
- 现代化深色界面，提供更好的视觉体验

## 特性

v2rayB 致力于提供最简单的操作，满足绝大部分需求。

得益于 Web 客户端的优势，你不仅可以将其用于本地计算机，还可以轻松地将它部署在路由器或 NAS 上。

## 使用方法

v2rayB 主要提供了下述使用方法：

1. 从 APT 软件源或者 AUR 安装
2. Docker
3. 从 GitHub releases 下载二进制与安装包

详见 [**v2rayA - Docs**](https://v2raya.org/docs/prologue/introduction/) 获取详细文档。
